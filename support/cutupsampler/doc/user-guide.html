Cutupsampler documentation

Overview

Cutupsampler is a program for Mac OS X that performs a cut-up of a sound file and maps the various cuts to keys on a MIDI keyboard or computer keyboard.

Quick start

  1. If you're using a MIDI controller (e.g. keyboard or other such device):
    1. Select an appropriate device from the "MIDI dev" menu.
    2. Select the MIDI channel on which the device will be transmitting, from the "MIDI ch" box.
    3. Adjust "# keys" to cut the audio sample file up into more or fewer chunks. each chunk will be mapped to a key/note of the MIDI controller. Adjust "low key" to define the key to use as the start of the total, overall sample.
  2. If you want to use the computer keyboard:
    1. In the "controller" panel, select "computer kbd".
  3. Select an appropriate audio output device from the "audio out" box.
  4. Click "load sound file" and select a WAV or AIFF audio file. If the file is monophonic, check the "file is mono" box.
  5. Play!

MIDI control

Computer keyboard control

The computer keyboard does not attempt to emulate a clavier keyboard; in other words, there are no black and white keys.

"Low key" box becomes irrelevant and is disabled. Use the "# keys" box to control the number of computer-keyboard keys into which the sound sample is divided. The number of active keys are counted from the top-left alphabetic key, going across and then down.

Known issues

Dragging mouse in dial while using the computer keyboard. Use space key to stop all held notes.

Computer keyboard only allows 6 notes at once. Not sure if this is the keyboard's circuitry or OS X, but that's the way it is.
